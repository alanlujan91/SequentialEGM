21The long-run multiplier for check stimulus is insensitive to the size of the check. The long-run multiplier for an $80 per capita stimulus check is 1.343 while it is 1.339 for a $1200 check. Hence, the downscaling of the stimulus check does not alter the impact on consumption per unit of policy expenditure. The same is true for the tax cut, where the change in the multiplier is almost zero.
